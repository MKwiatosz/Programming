### 2016/2017 - Kartkówka 1 - Grupa A - Zad. 1 (8 listopada 2016)

Mamy ciągi:
```
a(1) = 1;  
a(n) = a(n-1) + b(n)/3;  
b(1) = 2;  
b(n) = b(n-1) + a(n - 1);
```

Proszę napisać program, który wczytuje liczbę naturalną `k` i odnajduje wyraz należący do jednego z ciągów o wartości najbliższej k.
Program powinien wypisać numer znalezionego wyrazu i ciąg z którego on pochodzi. Jeżeli więcej niż jeden wyraz jest jednakowo odległy od k, należy wybrać ten o mniejszym numerze.
